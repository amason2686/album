
APPLICATION PURPOSE
Create a console application that displays photo ids and titles in an album. The photos are available in this online web
service (https://jsonplaceholder.typicode.com/photos).

This is a project using java that sends an http request to a webpage, takes the response from that webpage and puts it into an array, then outputs the result with the relevant fields, ie. albumId, Id, and title.

TO BUILD

Make sure you have maven installed, set the classpath to https-album

RUNNING

Pressing run will output a list of every albumId, Id, and title.
